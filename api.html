Make your Cocktail

Make Your Cocktail is a dynamic full-stack web application that empowers users to explore cocktail recipes either randomly or through a custom search. The project features responsive UI design crafted entirely with pure CSS, showcasing meticulous layout control and adaptability across devices. On the server side, EJS templates are used for dynamic rendering, while Axios handles API-driven data fetching from TheCocktailDB. This application demonstrates a clean architecture, interactive user experience, and effective integration of third-party APIs—without relying on any external CSS frameworks.

Tools & Technologies Used:
  • Built responsive layout entirely with Vanilla CSS using Flexbox, Grid, and media queries
  • Used EJS (Embedded JavaScript Templates) for server-side rendering of dynamic content
  • Styled custom scrollbars and UI components without any CSS libraries or frameworks
  • Created interactive buttons and hover animations using pure CSS transitions
  • Implemented parallax background effects and visual depth through layered styling
  • Crafted adaptive layout behaviors that respond to vertical shrinkage without distortion
  • Developed back-end with Node.js and Express.js for routing and server logic
  • Integrated Axios to fetch cocktail data asynchronously from external APIs
  • Used Body-Parser middleware to handle HTML form submissions and POST data
  • Fetched and rendered cocktail recipes using TheCocktailDB API via:
    • /random.php – to fetch a random drink
    • /search.php?s=drinkName – to search drinks by name
