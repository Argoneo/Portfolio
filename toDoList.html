TO-DO-LIST

To-Do-List is a full-stack to-do list web application that allows users to add, edit, and delete tasks in real time, with all data persistently stored in a PostgreSQL database. The app is built using Express and EJS on the server side, with dynamic rendering of task items and intuitive interactions in the browser. Styled with custom CSS and JavaScript, it offers a simple, elegant, and responsive user experience—without relying on frontend frameworks or component libraries

Tools & Technologies Used:
  • Node.js and Express.js for server logic and HTTP routing
  • PostgreSQL database connected via the official pgNode client
  • dotenv, for secure environment variable management (.env)
  • EJS (Embedded JavaScript Templates) for server-side rendering of dynamic task lists
  • body-parser middleware to handle POST form submissions cleanly
  • Fully functional CRUD operations (Create, Read, Update, Delete) implemented through Express routes
  • Vanilla CSS for all UI styling—no frameworks like Bootstrap or Tailwind
  • Responsive layout with media queries and a clean, mobile-friendly design
  • Styled input forms, buttons, and task rows with custom transitions and hover effects
  • JavaScript used on the frontend for inline task editing toggles and UI logic
  • Designed with a modular file structure and EJS partials for reusability (header, footer)
  • Task data fully persisted in the database (no local storage or in-memory reliance)
