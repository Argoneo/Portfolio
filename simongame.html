Simon Game

Simon Game is a browser-based memory game inspired by the classic electronic game of the same name. This interactive project challenges players to repeat an ever-growing sequence of colored button presses. With a nostalgic pixelated aesthetic powered by retro fonts and pure CSS, the game combines playful visuals with smooth gameplay logic. The implementation emphasizes DOM manipulation, audio feedback, animations, and responsive layout, making it engaging on both desktop and mobile devices.

Tools & Technologies Used:
  • Created the entire interface with semantic HTML5 and a minimalist layout
  • Styled the game using Vanilla CSSincluding responsive media queries for different screen sizes
  • Implemented full game logic in JavaScript, including:
    • Randomized pattern generation
    • Click-based user input tracking
    • Level progression logic
    • Game-over and reset handling
  • Enhanced user experience with CSS animations (pressed effect, game-over feedback)
  • Managed sound effects dynamically with the Audio object for real-time button feedback
  • Applied jQuery for DOM manipulation and event binding for concise, readable logic
  • Designed a mobile-friendly layout with scaling button sizes and adaptive text
  • All assets (sounds, visuals) handled locally—no external dependencies besides jQuery and fonts
