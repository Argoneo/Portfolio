Art Gallery

Modern Art Gallery is a fully responsive front-end website designed to deliver an elegant and immersive visual experience. The project showcases a striking landing page with a three-part layout—featuring black, white, and photographic sections—that introduces users to the world of modern art through refined typography, high-resolution imagery, and a focused call-to-action. A dedicated Location page offers interactive map details and address information, styled consistently with the homepage. The entire site is crafted without the use of any CSS frameworks, relying solely on Vanilla HTML, CSS, and a touch of JavaScript to implement smooth UI interactions and layout responsiveness.

Tools & Technologies Used:
  • Structured with semantic HTML5 for clean, accessible markup
  • Crafted a fully responsive layout using Vanilla CSS with Flexbox, Grid, and media queries
  • Built an interactive call-to-action button using vanilla JavaScript for hover animations
  • Integrated multiple Google Fonts with preconnect for optimized font loading and rendering
  • Used retina-ready (@2x) imagery for high visual fidelity on modern displays
  • Designed entirely from scratch without any UI libraries or frameworks like Bootstrap or Tailwind
  • Implemented dark/light logo switching for visual consistency across different sections
  • Developed a separate Location page with flexible layout and mobile-optimized navigation, including a dynamic “Back to Home” button
  • Added a fully styled footer with accessible social media icons (SVG) and gallery info
