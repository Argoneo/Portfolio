Using Bootstrap,
to create a home-restaurant-social website


Easy

"Easy" is a website created for an imaginary home-based restaurant.
The purpose was to demonstrate my knowledge of the Bootstrap framework by customizing parts of the code to make the site more user-friendly in terms of both UX and UI.

Changes and customizations include:
  • Used Bootstrap’s grid system and components to create a responsive and clean layout
  • Customized Bootstrap styles with my own CSS for colors, fonts, and spacing
  • Enhanced navigation bar with custom hover effects and smooth transitions
  • Implemented fixed background image with parallax effect for better visual appeal
  • Created custom media queries to hide elements on vertical shrink instead of resizing
  • Added interactive buttons with hover animations and shadow effects
  • Optimized carousel images with flexbox and object-fit for consistent aspect ratios
  • Structured footer with adaptive layout and input usability across devices
  • Maintained clean modular CSS separate from Bootstrap core files
