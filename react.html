Note-Book

Note-Book App is a dynamic, single-page note-taking application built with React and styled using both custom CSS and Material UI components. It allows users to create, view, and delete notes instantly in a clean, responsive layout. Designed with a focus on user interaction and component modularity, the app offers smooth animations, intuitive state handling, and a polished visual design that mimics the functionality of popular productivity tools like Google Keep.

Tools & Technologies Used:
  • React for component-based architecture and interactive UI
  • Utilized React Hooks (useState) for state management of notes and UI behavior
  • Material UI components and icons for floating buttons, transitions, and interaction feedback
  • Designed an expanding input interface using Zoom animation to enhance UX
  • Modularized code with clearly separated components: App, Note, Header, Footer, and CreateArea
  • Fully styled with Vanilla CSS for layout, color themes, and responsiveness
  • Google Fonts integration (Montserrat, McLaren) for modern and playful typography
  • Responsive and accessible design: mobile-friendly note cards with floating delete buttons
  • Background texture applied via external pattern URL for subtle visual depth
  • Deployed as a static SPA — ready for Netlify, Render, Vercel, or GitHub Pages
